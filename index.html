Latest Project

Gym Diet Suggester

Gym Diet Suggester is a smart web-based application designed to provide personalized diet plans for fitness enthusiasts. By analyzing user input such as body type, fitness goals (e.g., muscle gain, fat loss, maintenance), and activity levels, the system recommends a tailored diet chart with balanced macronutrients. Built using modern web technologies, it aims to promote healthy eating habits and support users on their fitness journey with actionable, data-driven suggestions.

Weather Forecasting App

Weather Forecasting App is a dynamic web application that provides real-time weather updates for any location worldwide. Using APIs like OpenWeatherMap, it fetches and displays current conditions including temperature, humidity, wind speed, and a multi-day forecast. The interface is user-friendly, responsive, and visually engaging, making it easy for users to stay informed about weather trends anytime, anywhere..

Transport Management System

Transport Management System is a web-based application designed to streamline and automate the logistics and transportation process. It allows efficient management of vehicle fleets, routes, drivers, and delivery schedules. With features like real-time tracking, trip assignments, and performance analytics, this system enhances operational efficiency, reduces delays, and ensures smoother coordination between transport administrators and users.

Pizza Restaurant Website

Pizza Restaurant Website is a responsive and user-friendly web application designed to showcase a modern dining experience. It includes interactive sections like a dynamic menu, an image gallery, an embedded location map, and an engaging carousel. Built with semantic HTML, CSS, and Bootstrap, it ensures accessibility, mobile responsiveness, and seamless navigation—offering customers an inviting way to explore and connect with the restaurant online.

Insta Clone

Insta Clone is a social media web application inspired by Instagram, allowing users to create accounts, upload images, like and comment on posts, and follow other users. Built with React for the frontend and Node.js with MongoDB for the backend, it replicates core social networking features while maintaining responsive design, user authentication, and real-time updates for a smooth and engaging user experience.

Traffic Rules Awareness System

Traffic Rules Awareness System is an educational web platform designed to promote road safety by educating users about essential traffic rules, signals, and safe driving practices. It features interactive quizzes, visual guides, and real-world scenarios to help users—especially learners and new drivers—understand and follow traffic regulations effectively. Built with a user-friendly interface, the system aims to reduce road accidents by increasing public awareness and responsibility.
